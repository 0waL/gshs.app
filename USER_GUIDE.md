# GSHS.app 사용자 안내문

이 문서는 실제 사용자 관점에서 GSHS.app 사용 방법을 간단하고 명확하게 정리한 안내문입니다.  
대상에 따라 필요한 기능이 다르므로 아래 3가지 경우로 나누어 읽어주세요.

- 학생
- 방송부
- 교사

공통 접속 주소:

- 운영 서비스: <https://gshs.app>
- 테스트 서비스: <https://test.gshs.app>

문의:

- 일반 문의/기능 제안: `admin@gshs.app`
- 오류 신고: 서비스 내 `/report`

---

## 1. 학생 안내

### 1-1. 처음 시작할 때

학생 계정은 초대 토큰을 받아 가입하는 방식입니다.

가입 방법:

1. 학교에서 안내받은 초대 메일 또는 토큰을 확인합니다.
2. 회원가입 링크(`/signup?token=...`)로 들어갑니다.
3. 이름, 아이디, 비밀번호 등 가입 정보를 입력합니다.
4. 가입을 마치면 로그인 후 개인 기능을 사용할 수 있습니다.

만약 아직 초대 메일을 받지 못했다면:

- 학교에서 별도로 안내된 토큰 배부 포털을 사용하거나
- 관리자에게 직접 문의해 초대 메일을 요청하세요.

### 1-2. 학생이 사용할 수 있는 주요 기능

학생 계정으로 로그인하면 아래 기능을 사용할 수 있습니다.

- 홈: 오늘 확인해야 할 정보 한눈에 보기
- 공지사항: 학교 공지 열람
- 급식: 오늘/다음 급식 확인
- 시간표: 학년/반 기준 시간표 확인
- 학사일정: 학교 일정 확인
- 링크모음: 자주 쓰는 학교 관련 링크 확인
- 교내 사이트: 로그인 후 학교 내부 사이트 바로가기 사용
- 기상곡 신청: 유튜브 링크 기반 신청
- 내 정보: 계정 정보, 개인 일정, 비밀번호 변경
- 알림: 시스템/기능 관련 알림 확인
- 오류 신고: 사용 중 문제를 관리자에게 전달

### 1-3. 학생이 자주 묻는 상황

#### 기상곡은 누가 신청할 수 있나요?

학생은 기상곡 신청 페이지에서 유튜브 링크를 제출할 수 있습니다.  
요일별 신청 제한이 있을 수 있으며, 그날 신청 가능한 학년이 아니면 신청 폼이 제한됩니다.

#### 개인 일정이 없으면 어떻게 보이나요?

홈 화면과 내 정보 페이지에서 개인 일정이 없을 경우 일정 없음 안내가 표시됩니다.  
필요하면 내 정보에서 직접 추가할 수 있습니다.

#### 교내 사이트가 안 열리면 왜 그런가요?

교내 사이트는 로그인한 사용자만 접근할 수 있습니다.  
로그인하지 않은 상태라면 먼저 로그인 후 다시 시도하세요.

#### 비밀번호를 잊어버렸어요

관리자에게 문의해 비밀번호 초기화를 요청하세요.  
현재 비밀번호 초기화는 관리자 권한에서만 가능합니다.

### 1-4. 학생용 핵심 주의 사항

- 회원가입 토큰은 다른 사람과 공유하지 마세요.
- 초대 토큰은 유효 기간이 지나면 다시 발급받아야 합니다.
- 기상곡 신청은 검수 후 승인/반려될 수 있습니다.
- 계정 문제가 있으면 스스로 데이터 수정 시도하지 말고 관리자에게 문의하세요.

## 2. 방송부 안내

### 2-1. 방송부 권한은 어떻게 받나요?

방송부는 별도 계정을 새로 만드는 방식이 아니라,  
기존 학생 계정에 방송부 권한(`BROADCAST`)을 추가로 부여하는 구조입니다.

즉:

- 먼저 일반 학생 계정으로 가입/로그인하고
- 이후 관리자가 해당 계정에 방송부 권한을 부여합니다

권한 부여가 완료되면, 같은 계정으로 방송부 기능을 사용할 수 있습니다.

### 2-2. 방송부가 사용할 수 있는 기능

방송부는 학생 기능을 그대로 사용하면서 아래 기능이 추가됩니다.

- 방송부 페이지(`/music`) 접근
- 기상곡 신청 목록 검토
- 승인 / 반려 처리
- 필요한 경우 반려 사유 기록
- 방송 진행에 필요한 곡 상태 관리

### 2-3. 방송부 업무 흐름

일반적인 사용 흐름:

1. 로그인
2. 방송부 페이지 진입
3. 대기 중인 기상곡 신청 확인
4. 제목, 썸네일, 링크, 신청자 정보를 검토
5. 승인 또는 반려
6. 필요 시 재생 완료 상태까지 관리

### 2-4. 방송부용 주의 사항

- 방송부 권한은 학생 메타데이터를 유지한 채 부여됩니다.
- 방송부 계정은 여전히 개인 계정이므로 비밀번호 공유를 권장하지 않습니다.
- 반려 시에는 가능한 한 신청자가 이해할 수 있도록 사유를 남기는 것이 좋습니다.
- 실수로 잘못 처리했거나 접근 권한이 이상하면 관리자에게 바로 문의하세요.

### 2-5. 권한이 안 보일 때

아래 경우에는 방송부 기능이 보이지 않을 수 있습니다.

- 아직 관리자 권한 변경이 반영되지 않은 경우
- 로그인 세션이 오래되어 권한 갱신이 안 된 경우

이 경우에는:

1. 로그아웃
2. 다시 로그인
3. 그래도 안 보이면 관리자에게 문의

순서로 확인하세요.

## 3. 교사 안내

### 3-1. 교사 계정은 어떻게 받나요?

교사 계정은 공개 포털에서 직접 신청하는 방식이 아니라,  
관리자가 초대 토큰 또는 초대 메일을 직접 발송하는 방식으로 운영합니다.

교사 계정이 필요하다면:

- 관리자에게 계정 생성 또는 초대 메일 발송을 요청하세요.

### 3-2. 교사가 사용할 수 있는 주요 기능

교사는 일반 로그인 사용자 기능에 더해 아래 기능을 사용할 수 있습니다.

- 링크모음 관리
- 공지 작성 진입
- 일반 공지 열람, 급식, 일정, 시간표 등 공통 기능 사용
- 내 정보, 알림, 오류 신고 사용

### 3-3. 교사에게 중요한 기능

#### 링크모음 관리

교사는 필요한 학교 관련 링크를 추가, 수정, 삭제할 수 있습니다.  
이 기능은 링크모음 페이지의 편집 권한으로 제공됩니다.

#### 공지 작성

교사는 공지 작성 진입점을 통해 필요한 공지를 게시할 수 있습니다.  
공지 작성 권한은 일반 학생에게는 제공되지 않습니다.

### 3-4. 교사용 주의 사항

- 교사 권한으로 변경되면 학생 메타데이터(학번, 기수)는 제거될 수 있습니다.
- 학생 계정과 교사 계정은 역할과 접근 범위가 다르므로, 권한 변경은 관리자에게 요청하세요.
- 공지나 링크를 수정하기 전에는 실제 서비스에 노출되는 내용을 다시 확인하세요.

## 공통 안내

### 서비스에서 자주 보는 정보 출처

주요 정보는 아래 기준으로 제공됩니다.

- 급식: NEIS Open API 기반
- 학사일정: 내부 일정 + 외부 캘린더 연동 + 학교 일정 데이터
- 공지사항: 관리자 작성 내부 데이터

정확한 설명은 서비스 내 `/help`에서 확인할 수 있습니다.

### 문제 발생 시

상황별 권장 경로:

- 단순 문의 / 기능 제안: `/help` 또는 `admin@gshs.app`
- 실제 오류 / 버그 신고: `/report`
- 계정 권한 문제: 관리자에게 직접 문의

### 마지막 안내

GSHS.app은 학교 구성원이 자주 쓰는 기능을 한곳에 모으는 서비스입니다.  
권한별 기능은 계속 개선될 수 있으므로, 화면 구성이나 버튼 위치가 조금 바뀌더라도 기본 사용 흐름은 위 문서를 기준으로 이해하시면 됩니다.
